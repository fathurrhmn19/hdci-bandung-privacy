FAQ
Pertanyaan Umum
Saya tidak menerima kode OTP WhatsApp
Pastikan nomor HP yang Anda masukkan aktif dan terdaftar di WhatsApp. OTP berlaku selama 5 menit. Jika tidak diterima dalam 2 menit, coba lagi. Jika masih gagal, hubungi admin melalui tombol Bantuan di aplikasi atau WhatsApp di atas.
Status pendaftaran saya sudah lama masih "Menunggu Review"
Proses review formulir pendaftaran biasanya membutuhkan waktu 1–3 hari kerja. Jika sudah lebih dari 3 hari, hubungi admin via WhatsApp dengan menyebutkan nama lengkap dan nomor HP yang digunakan saat mendaftar.
Saya ingin mengubah data profil (nama, alamat, kendaraan)
Buka aplikasi → Profil → Edit Profil. Beberapa perubahan data sensitif (seperti foto KTP) mungkin memerlukan persetujuan admin. Untuk perubahan nomor HP, hubungi admin langsung.
Pembayaran iuran/event saya sudah transfer tapi belum diverifikasi
Verifikasi pembayaran dilakukan secara manual oleh admin, biasanya dalam 1×24 jam kerja. Pastikan Anda sudah mengunggah bukti transfer di aplikasi. Jika lebih dari 24 jam belum diproses, hubungi admin via WhatsApp dengan menyertakan foto bukti transfer.
Saya ingin menghapus akun atau data saya
Hubungi admin via WhatsApp 0812 3450 0176 dengan menyebutkan nama lengkap dan nomor HP terdaftar, lalu minta penghapusan akun. Data akan dihapus dalam 30 hari kerja sesuai Kebijakan Privasi kami.
Aplikasi crash / tidak bisa dibuka
Coba tutup aplikasi sepenuhnya lalu buka kembali. Pastikan aplikasi sudah versi terbaru dari Play Store / App Store. Jika masih bermasalah, screenshot pesan error-nya dan kirim ke admin via WhatsApp.
Bagaimana cara melaporkan pelanggaran privasi atau keamanan data?
Hubungi admin via WhatsApp 0812 3450 0176 dan jelaskan pelanggaran yang terjadi. Kami akan merespons dalam 14 hari kerja sesuai UU PDP No. 27 Tahun 2022.
